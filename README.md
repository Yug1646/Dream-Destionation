# Dream-Destionation

A simple static HTML page showcasing a dream travel destination. The page highlights the Golden Gate Bridge, top attractions, travel plans, and useful links for visitors.

## Files

- `index.html` — Main HTML page for the travel destination showcase.
- `ggb.jpg` — Image shown on the page (if present in the workspace).

## Features

- Title and heading for the travel destination.
- Description of the Golden Gate Bridge.
- Image section for the destination.
- List of top attractions.
- Ordered list of things to do.
- Travel plan table with cities, estimated cost, and days.
- Useful links for more information and support.

## How to View

1. Open `index.html` in a web browser.
2. Alternatively, use a local development server to preview the page.

## Notes

- This is a static HTML-only project.
- No JavaScript or CSS files are required for the basic layout.
- Update `index.html` to change content or add styling.
